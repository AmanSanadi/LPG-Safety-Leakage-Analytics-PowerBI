# 🔥 LPG Cylinder Safety & Leakage Analytics

> A multi-page Power BI dashboard project for monitoring LPG cylinder operations — tracking complaints, leakage rates, inspection quality, and operational costs across plants and zones.

## 📌 Overview

This project analyses LPG cylinder distribution operations across **three manufacturing plants** (Belagavi, Hubballi, Mangalore) and **four geographic zones** over the period **2024–2026**.

It addresses key safety and operational risks:
- LPG leakage and defective cylinders
- High complaint volumes and delayed resolutions
- Inspection failures and quality gaps
- Rising operational costs

## ✅ Requirements

To open and explore this project you will need:

- **Power BI Desktop** (latest version recommended) — [Download here](https://powerbi.microsoft.com/desktop)
- **Microsoft Excel** or any compatible tool to view the source dataset
- Basic familiarity with DAX for exploring calculated measures

---

## 🚀 Getting Started

1. Clone or download this repository
2. Open `dashboard/LPG_Analytics.pbix` in Power BI Desktop
3. Connect to the source data in the `data/` folder if prompted
4. Use the slicers (Year, Plant, Zone) to explore the dashboards interactively
